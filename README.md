## 專案簡介

### 遊戲簡介

《長號冠軍》是一款節奏遊戲，玩家使用虛擬長號演奏各種歌曲，從古典和公共領域歌曲到原創曲目和混音。演奏者移動滑鼠來改變長號的音高，並使用滑鼠按鈕或鍵盤按鍵來演奏。玩家通過在螢幕上滑動時成功敲擊音符來賺取積分，通過保持連續的連擊並填滿計量表來獲得更多積分。

## 遊戲介紹

### 遊戲規則

利用滑鼠控制音高並在音符抵達線上後按下滑鼠左右鍵、wasd鍵或zxc鍵來判定。  

如果遇到一顆音符需要短按，遇到長音符則需要長按。  

按完音符後會產生五種判定，分別為:PERFECTO, NICE, OK, MEH, NASTY。  

在遊玩過程中想要重新開始可以長按R重新開始，也可以按esc選擇需要返回、重新開始或回到選單  

依據不同的準確度會給予1000、800、500、200、0的分數，當整首歌玩完後會顯示分數及哪一種判定出現了幾次。  

最後會依據整體打擊來給予F到S不等的評分供查看。

## 程式設計
### 程式架構

將程式分為四大區塊
1. `app.hpp`設計遊戲的五種狀態，控制每個階段。例如在START階段把所有物件初始化並讀取歌檔，並隨著狀態切換來更新遊戲邏輯、暫停或切換到結算畫面。
2. `keyboard.hpp` 偵測玩家的滑鼠與鍵盤輸入（支援滑鼠點擊、WASD、ZXC 鍵）。可以判斷玩家對音符是短按還是長按，同時建立了防重複觸發的機制。
3. `note.hpp`負責音符的動態生成，將`app.hpp`讀進來的譜面資料，偵測該音符是不是彎曲長條音符。是則把該段音符的所有座標檔案全部讀取完畢，再完整生成為會轉彎的單一長音符。
4. `ScoreManager.hpp`負責計分與統計。音符被判定後依據準確度即時累加分數，更新當前連擊數並記錄最高連擊。整首歌結束時統計出所有判定出現的次數，並計算出最終的評級傳給結算畫面。

### 程式技術

使用smart pointer解決可能的memory leak問題，當音符跑向最左側出了畫面以後安全釋放記憶體。    

利用程式與數學結合將本來利用畢氏定理轉換方向的單調音符變為圓滑的音符，同時避免程式把單一轉彎音符誤讀為三顆音符。  

使用weak_ptr，當畫面出現多顆音符時，所有音符共享同一個記憶體中的長條圖檔。當歌曲結束時將圖片資源釋放。  

使用強型別列舉與有線狀態機確保切換狀態時沒有問題。
